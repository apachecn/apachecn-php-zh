# 十九、附录 B：网关前的代码

本附录显示了遗留应用程序的部分页面脚本。它已从实际应用程序中经过消毒和匿名处理。

这个脚本是一个系统的一部分，该系统允许新闻专业的学生撰写文章进行评论，并对同学的文章提供反馈。学生将为其他学生提供“学分”来复习他们的作品，并通过复习他们的文章获得其他学生的学分。因为每次复习都要支付学分，学生们会限制复习的最大次数，以确保他们不会用完学分。最后，他们被允许提供注释，指出评审员应该注意什么。

这是转换为使用网关类之前的页面脚本版本。它只包含领域逻辑和数据源交互，不包含初步设置或任何显示代码。
