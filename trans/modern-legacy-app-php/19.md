# 附录 C.网关后的代码

本附录显示了从附录 B 转换为使用网关类后的页面脚本版本。请注意，它的变化非常小。即使删除了 SQL 语句，域业务逻辑仍然嵌入在页面脚本中。

网关类在页面脚本下面提供，并显示到 PDO 样式绑定参数的转换。还要注意，页面脚本中的`if()`条件有一些小的修改：以前他们检查查询是否成功，现在他们检查来自网关的返回值。
